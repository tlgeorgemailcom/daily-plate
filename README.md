A daily word chain puzzle game using food words. Build a chain where each word starts with the last letter of the previous word!

## How to Play

1. Start with the daily starting word
2. Add 5 food words to complete the chain
3. Each word must start with the last letter of the previous word
4. Score points based on word length (+10 bonus for completing!)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
